Collect Crystals Games

Inspiration

The idea of Collect Crystals began when I had the desire to create a simple Game with the coding knowledge I had. Due to the limitations of my skills I decided to create a terminal based game. I had the idea of updating a 2d array of ascii characters in a turn based fashion based on keypress events executed by the player on their keyboard

1st Iteration (JAVA)

Github to Java code

The first iteration of Collect Crystals was created using objects in java.

I used a Key Listner library to implement player movement using arrow keys, however it was inconvenient to use because a window had to be open and in focus above the terminal window to work. All the crystal and enemy spawn and position information was kept in this class to so it could be modified on key presses.

The game field features "< >" as enemies, "**" as the crystals and "o7" or the symbol for a man saluting as the player character.

2nd Iteration (PYTHON)

Github to Python code

For my first recreation of Collect Crystals I wanted to make it better than the first version somehow. I searched for a way to create a UI for the game rather than having to run it in the terminal to which I stumbled upon the tkinter library. Essentially I drew images in a tkinter window for each cell of the grid and updated them in a similar fashion to the first iteration of the game.

A square class was used which holds the state of a square whether that be an enemy (RED), crystal (BLUE), player (GREEN) or standard game square (WHITE).
